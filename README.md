# Kube-Prometheus Stack Automation

Atlantis for diploma project

## Secrets Required in GitHub:
- `YC_TOKEN` - Yandex Cloud OAuth or IAM токен
- `YC_CLOUD_ID`
- `YC_FOLDER_ID`
- `YC_ACCESS_KEY` - Static key for S3
- `YC_SECRET_KEY` - Secret key for S3 
- `YC_S3_BUCKET` - bucket name
- `YC_S3_STATE_KEY` - state key file name
- `KUBE_CONFIG_DATA` - kubeconfig
- `GRAFANA_ADMIN_PASSWORD`
