# Kube-Prometheus Stack Automation

## Secrets Required in GitHub:
- `YC_TOKEN` - Yandex Cloud OAuth token
- `YC_CLOUD_ID` 
- `YC_FOLDER_ID` 
- `YC_ACCESS_KEY`
- `YC_SECRET_KEY` 
- `YC_S3_BUCKET` - bucket name
- `YC_S3_STATE_KEY` - state file name
- `KUBE_CONFIG_DATA`
- `GRAFANA_ADMIN_PASSWORD`
